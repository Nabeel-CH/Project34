\chapter{Software and Dependencies}

All code is implemented in Python. The key libraries used are:

\begin{itemize}
    \item \textbf{pydicom} \cite{pydicom2024} --- DICOM file reading.
    \item \textbf{OpenCV} \cite{bradski2000opencv} --- image processing (thresholding, morphology, CLAHE, resizing).
    \item \textbf{Kymatio} \cite{andreux2020kymatio} --- wavelet scattering transform computation.
    \item \textbf{scikit-learn} \cite{pedregosa2011scikit} --- k-NN classifier, cross-validation, StandardScaler, evaluation metrics.
    \item \textbf{NumPy, Pandas, Matplotlib} --- numerical computation, data handling, and visualisation.
\end{itemize}

The vendored \texttt{scattering\_transform} package (Cheng \& M\'enard) is imported locally for the scattering-covariance experiments and does not require installation. Exploratory, negative-control, and superseded methods that were tried but not adopted are catalogued separately in Appendix~\ref{app:negative_controls}.
